
## 1. Running the server 

```bash
  cd server
```

Install dependencies

```bash
  npm install
```

Starting the server

```bash
  npm run aezakmi
```
**or**
```bash
  node .\server.js
```

## 2. Running the React.js Project 

```bash
  cd login-jwt-1c
```

Install dependencies

```bash
  npm install
```

Starting the react.js Project

```bash
  npm run hesoyam
```
**or**
```bash
  npm start
```
